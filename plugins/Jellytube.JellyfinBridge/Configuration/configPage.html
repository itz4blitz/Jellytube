Jellytube Bridge

Configure the public Jellytube endpoint that Jellyfin should hand authenticated users off to for web video requests.
The public Jellytube origin users should land on after the Jellyfin handoff.
The Jellytube route that receives signed handoff tokens.
Must match the Jellytube service's HANDOFF_SECRET value.
Short-lived token expiry used for the handoff redirect.
Fallback path inside Jellytube when no explicit target is supplied.
Recommended user launch URL inside Jellyfin Web: /JellytubeBridge/start
Browser extension launch URL example: https://your-jellyfin.example.com/JellytubeBridge/start
